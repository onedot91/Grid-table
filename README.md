# Grid Table

Serverless match tracker app built with Vite + React.
All match data is stored in browser `localStorage`.
Records persist across refreshes and are reset only by the `RESET` button.

## Run Locally

Prerequisites: Node.js

1. Install dependencies:
   `npm install`
2. Start dev server:
   `npm run dev`
3. Build for production:
   `npm run build`
